# Postoperative Spinal Infection Risk Calculator

Streamlit Community Cloud entry point:

```text
spinal_infection_public_app_v3/app.py
```

The app files, model artifact, scaler, metadata, and Python dependencies are all in `spinal_infection_public_app_v3/`.

## Deploy

1. Push this repository to GitHub.
2. Go to https://share.streamlit.io/deploy.
3. Choose the GitHub repository and branch.
4. Use `spinal_infection_public_app_v3/app.py` as the main file path.
5. Click deploy.

## Run Locally

```bash
cd spinal_infection_public_app_v3
pip install -r requirements.txt
streamlit run app.py
```
